# 강익준 | Game Server Programmer

안녕하세요. 다양한 장르와 플랫폼의 라이브 서비스를 경험한 게임 서버 개발자 강익준입니다.

MMORPG, 수집형 RPG, SNG, 블록체인 게임을 거치며 **안정적인 서비스 운영**, **글로벌 플랫폼 연동**, **확장 가능한 서버 구조**를 중심으로 일해왔습니다.  
기능을 만드는 것에서 끝내지 않고, 운영 중에 문제가 빨리 보이고 다시 고치기 쉬운 구조를 좋아합니다.

## What I Work On

- 게임 서버 아키텍처 설계와 구현
- 글로벌 로그인, 결제, 플랫폼 연동
- Redis 기반 캐시, 세션, 로그 파이프라인
- AWS/GCP 기반 클라우드 서버 운영
- 라이브 서비스 안정화와 장애 대응

## Featured Experience

### Re:Memento

신규 글로벌 프로젝트의 서버 프로그래머로 참여했습니다.

- C++, C#, .NET, MySQL, MongoDB, Redis, Protobuf, AWS EKS 사용
- 프로젝트 초기 단계부터 서버 아키텍처 설계와 구현 참여
- 글로벌 통합 로그인, IAP, Payletter, Epic Store, VNG 등 외부 플랫폼 연동
- Redis Stream 기반 실시간 서버 로그 수집 파이프라인 구축
- 장애 대응 속도를 높이기 위한 로그 수집, 분석 흐름 개선

### Blockchain Game Service

프로그래밍 실장으로 Web3 백엔드와 서버리스 인프라를 함께 담당했습니다.

- Node.js, TypeScript, Solidity, AWS Lambda, GCP Functions 사용
- 스마트 컨트랙트와 게임 서비스 백엔드 연동
- 서버리스 구조 도입으로 오토스케일링 가능한 운영 환경 구성
- 레거시 PHP, MySQL, Redis 기반 서비스 안정 운영

### MMORPG Server

MMORPG 서버 개발자로 대규모 필드와 실시간 동기화 로직을 구현했습니다.

- Node.js, MySQL, Redis, Linux 사용
- AOI 기반 관심 영역 처리 구현
- 월드, 인스턴스 던전, 충돌 판정, 동기화 로직 개발
- 대규모 필드 메시지 처리량을 고려한 서버 구조 개선

### I Love Coffee

장기 라이브 서비스의 클라이언트와 서버 개발을 함께 담당했습니다.

- C++, Lua, Cocos2d-x, Java, Unity, MySQL, MongoDB, Redis 사용
- 라이브 서비스 안정화와 신규 콘텐츠 업데이트
- 신규 과금 시스템 설계와 개발
- CS/QA 운영 도구 개발 및 LINE 플랫폼 이슈 대응

## How I Work

- 먼저 서비스 요구와 운영 흐름을 이해한 뒤 구현합니다.
- 새로운 기술을 도입할 때는 유지보수 비용과 팀의 학습 비용을 함께 봅니다.
- 장애가 났을 때 원인을 빨리 찾을 수 있는 로그와 관측 가능성을 중요하게 생각합니다.
- 추상화보다 읽기 쉬운 코드를 선호하고, 프로젝트에 맞는 단순한 구조를 지향합니다.
